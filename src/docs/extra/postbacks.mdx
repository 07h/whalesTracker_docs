Postback - механизм взаимодействия между партнерской программой и Вашим трекером.

Каждый переход по ссылке сохраняет свой **subid**, который необходимо передать в Вашу парнерскую программу. Если человек совершит покупку, то ПП отправит Get запрос с subid и с суммой оплаты.

У каждой ПП свой метод получения subid, поэтому стоит уточнить у Вашего менеджера в пп, как правильно передавать subid. 

## Ссылка для ПП

```
http://urlpp.com/?u=gfd3fds&o=5nd39f&cid=[subid]
```

Макрос в ссылке [subid] (**именно с квадратными скобками!**)

## Postback

```
http://ДОМЕНТРЕКЕРА/postback/__aw6dov?subid={cid}&payout={sum}
```
__aw6dov - ваш постбэк ключ. Ваше значение можно посмотреть и изменить в **Settings-General-Postback**

subid и payout - обязательные query ключи

{cid} и {sum} - макросы ПП. Ваши уточняйте у менеджера

> **Внимание** <br/> В настройках группы и потоков логирование должно быть включено, там где вы ожидаете перенаправление трафика в ПП и получение Postback
