На странице настроек представлено 2 вкладки.

## Основные

|name|default|information|
|--------|------|----------|
|Postback|random key|Ключ постбека|
|Clear statistics days|30|Сколько дней хранить статистику в БД|
|Clear remote days|25|Сколько дней хранить remote уводы в бд|
|Trash|Redirect on Url|Отправляет пользователя либо по ссылке либо показывает 404 ошибку|
|Trash URL|example.com|Ссылка для мусорного трафика|
|Query key|q|Ключ query для статистики и remote|
|Limit click|150|Сколько кликов показывать в таблице дашбоард|
|Limit amount|50|Сколько покупок показывать в таблице дашбоард|


## Другие

Представлена 3мя списками:
 - Список ip
 - Список сигнатур
 - Remote ссылки. Ссылки для увода

Для каждого списка можно выбрать следующие действия:

 - Edit - Редактируем текущий список
 - Add - Добавляем список к текущему списку
 - Delete - Удаляем список из текущего списка
 - Clear - Полностью очищаем список
