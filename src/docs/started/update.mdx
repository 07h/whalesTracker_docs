---
title: Обновление трекера
description: Обновление трекера на сервере.
---

Для обновления трекера необходимо выполнить несколько простых команд

1. Зайти на сервер по ssh

2. Зайти в папку, где лежат файлы трекера. Если Вы выполняли команды в точности, как на странице установки, то Ваша папка называется tds

    ```sh 
        cd tds
    ````

3. Выполните следующую команду

    ```sh 
        docker-compose down && docker-compose pull && docker-compose up -d --build && docker image prune -f
    ````

Эта команда остановит трекер, скачает обновления, потом перезапустит тдску и удалит старые образы.
Все настройки и данные переходов сохранятся, перенастраивать софт не придется. Проверьте только файлы Black sugnature, Remote url, List black IP они могут очиститься, поэтому перед обновлением предварительно сохраните эти данные где-нибудь
