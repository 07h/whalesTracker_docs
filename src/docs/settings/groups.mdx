Добавление новых групп происходит через кнопку **+** на странице Дашбоард. 

Страница группы имеет следующие элементы:

1. Настройку данной группы
2. Список привязанных к ней потоков, с возможностью их редактирования. Порядок срабатывания можно регулировать путем перетягиванием потоков между собой
3. После выбора любого из потоков. Отображаются связанные с ним настройки фильтров, порядок выполнения проверок которого, также регулируются перетягиванием.

## Свойства группы

|Name|Default|Information|
|----|-------|-----------|
|Label|None|Имя группы|
|Indentifier|random|Индификатор группы необходим для поиска группы при работе трекера|
|Type redirect|Http Redirect|Тип редиректа для перераспределения трафика|
|Input|None|Указываем ссылку, js, html код и другие значения характерные для выбранного Вами типа редиректа|
|Check unique|False|Проверять уникальность входящего трафика (Проверка ведется по Ip)|
|Hour unique|24|Через сколько часов повторный переход будет считаться уникальным|
|Active|True|Активная ли группа в текущий момент|
|Logging|True|Сохранять все переходы в текущей группе. Ограничить количество дней хранения можно через настройки|

> Индификатор группы не должен повторяться!
