Порядок перебора фильтров идет сверху вниз. Их можно свободно перетаскивать фильтры между собой, чтобы изменить порядор.

 - SmartTV

### Cities

Можно указать любую страну. Писать только латинницей. Нажимать enter в качестве разделителя. Например Moscow, San Antonio.

### Browsers languages

Язык браузера. Писать латинницей, enter в качестве разделителя. Пример ru-RU, ru, en-US, en

### Useragent includes

Юзераген браузера содежит какое либо значение. Пример Instagram, google bot

### Referrer includes

Рефер содержит какое либо значение. Пример google.com

### Referrer is Null

Рефер ничего не содержит

### Useragent is Null

Пустой юзерагент

### ipv6 mask

IPv6 содежит какое либо значение. Пример face

### Use list ips

Ip находиться в черном списке. Указать список ip можно в настройках во вкладке Другие

### Use list signature

Юзерагент содержит сигнатуру из черного списка. Указать список сигнатур можно в настройках во вкладке Другие


> **P.S.** Не забывайте сохранять все свои изменения!
