Позволяет забирать юзеров со страницы, которые подходят под определенные условия

Открывается iframe в котором отображается требуемая страница ссылка на которую была указана в настройках группы/потока.

> ВНИМАНИЕ <br/> Убедитесь, что указываемый сайт не имеет в хедерах следующую настройку <br/> X-Frame-Options: DENY <br/> Про данный хэдер Вы можете прочитать [Здесь](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/X-Frame-Options)

## Использование

1. Необходимо создать группу с типом редиректа Stop
2. Создайте потоки с необходимыми условиями. (Например, отбирать только мобильники)
3. Разместите код на той странице, на которой необходим отбор юзеров.

```
<script type="text/javascript" src="http://tracker.com/groupId"></script>
```

Теперь при заходе на страницу с мобильного юзеры будут отправлятся по ссылке указаной в настройках потока, а остальные, будут оставаться на этой же странице
