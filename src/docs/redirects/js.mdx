Редирект происходит через создание html страницы у которой в head прописан js script редиректа на ссылку

```js
window.location = "${url}"
```

## Использование

Пример использования в HTML для использования на сайте

```html
    <a href="http://tracker.com/group">Ссылка</a>
```
